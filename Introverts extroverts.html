What I Did

I transformed this raw behavioral data into a working personality prediction model using a supervised learning approach. Here's a quick snapshot of my workflow:

  • Data Cleaning: I handled missing values using median and mode imputation, depending on the feature type.
  • Encoding: Binary values like "Yes"/"No" were encoded into 1s and 0s.
  • Exploratory Data Analysis (EDA): I visualized relationships, like how "time spent alone" inversely correlated with extroversion.
  • Modeling: I tested multiple classifiers (Logistic Regression, Random Forest, and SVM), optimizing for accuracy and interpretability.
  • Evaluation: Achieved strong classification accuracy while keeping the model interpretable‚Äîideal for practical use cases like HR assessments, user personalization, or behavioral studies.
